Kammer 02 // Raumspalt-Kalibrierung

Zwei instabile Raumspalt-Kerne müssen an den richtigen Wandknoten befestigt werden. Ziehen Sie die Kerne auf die Knoten. Auf dem Handy können Sie einen Kern antippen und danach den Zielknoten antippen.

Kalibrierungsregeln:

1. Der rote Kern gehört tiefer als der grüne Kern.

2. Der grüne Kern liegt weiter rechts als der rote Kern.

3. Der rote Kern berührt die Startmarkierung.

4. Der grüne Kern liegt diagonal gegenüber dem roten Kern.

Das System weist darauf hin, dass Schrauben theoretisch wichtig sind. Praktisch wird das später entschieden.
